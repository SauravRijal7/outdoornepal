# Outdoor Nepal Printing

**Website:** [outdoornepal.com.np](https://outdoornepal.com.np)

Outdoor Nepal Printing provides high-quality printing services for businesses and individuals across Nepal. Our website offers a modern, user-friendly interface for exploring services, requesting quotes, and connecting directly with our team.

## Features

- **Modern React UI:** Sleek, responsive design for a smooth user experience on all devices  
- **EmailJS Integration:** Direct inquiry system without the need for a backend  
- **Service Showcase:** Detailed information on printing services, products, and solutions  
- **Responsive Design:** Optimized for desktops, tablets, and mobile devices  

## Tech Stack

- **Frontend:** React.js  
- **Styling:** Tailwind CSS / Custom CSS  
- **Hosting:** GitHub Pages 
- **Email Integration:** EmailJS for contact forms
